# Project Scope Statement

## Project Purpose
The purpose of this project is to create a simple personal webpage that introduces who I am 

## Description
This project involves building a single-page website using semantic HTML. The page will include my name, major or interests, two paragraphs describing my goals for the course or my career, an unordered list detailing my hobbies, and a link to my GitHub profile.

## Desired Results
- A working HTML webpage
- A page that's clearly structured, clean, perfectly indented, and displays a mastery of the concepts using semantic HTML elements
- A page that introduces me professionally

## Exclusions
- Advanced styling
- Visually stunning design
